### 1\. ‚è© Ph∆∞∆°ng ph√°p Kh√¥ng ƒê·ªá quy (Iterative / V√≤ng l·∫∑p)

ƒê√¢y l√† ph∆∞∆°ng ph√°p hi·ªáu qu·∫£ v√† ƒë∆∞·ª£c khuy√™n d√πng. Ch√∫ng ta t·∫°o m·ªôt m·∫£ng 2 chi·ªÅu (ho·∫∑c 1 chi·ªÅu) ƒë·ªÉ l∆∞u k·∫øt qu·∫£ c·ªßa h√†ng tr∆∞·ªõc ƒë√≥, v√† d√πng n√≥ ƒë·ªÉ t√≠nh h√†ng hi·ªán t·∫°i.

```cpp
/**
 * @brief In tam giac Pascal (Khong de quy - Iterative)
 * Su dung mang 2 chieu de luu tru.
 * @param n So dong can in
 */
void InPascal_Iterative(int n) {
    // Gia su n khong qua lon, dung long long de tranh tran so
    long long triangle[50][50]; 
    if (n > 50) {
        cout << "n qua lon, vui long giam n.\n";
        return;
    }

    // Duyet qua tung hang (i tu 0 den n-1)
    for (int i = 0; i < n; i++) {
        
        // In ra cac khoang trang de can le dau hang
        for (int k = 0; k < (n - 1 - i); k++) {
            cout << "   "; // Moi khoang trang = 3 ky tu
        }

        // Duyet qua tung phan tu trong hang i (j tu 0 den i)
        for (int j = 0; j <= i; j++) {
            
            // --- Day la phan co so (Base Case) ---
            // Neu la phan tu o canh (dau hang hoac cuoi hang)
            if (j == 0 || j == i) {
                triangle[i][j] = 1;
            }
            // --- Day la phan tinh toan ---
            // Nguoc lai, tinh theo cong thuc:
            // So hien tai = (so o tren-trai) + (so o tren-phai)
            else {
                triangle[i][j] = triangle[i-1][j-1] + triangle[i-1][j];
            }
            
            // In gia tri, setw(6) de can le cac cot
            cout << setw(6) << triangle[i][j];
        }
        cout << endl; // Xuong dong khi in xong 1 hang
    }
}
```

### 2\. üîÑ Ph∆∞∆°ng ph√°p ƒê·ªá quy (Recursive)

**C·∫£nh b√°o:** Gi·ªëng nh∆∞ h√†m Fibonacci ƒë·ªá quy, h√†m n√†y **c·ª±c k·ª≥ ch·∫≠m** khi `n` l·ªõn (v√≠ d·ª• `n > 20`) v√¨ n√≥ ph·∫£i t√≠nh to√°n l·∫∑p l·∫°i c√°c gi√° tr·ªã r·∫•t nhi·ªÅu l·∫ßn (v·∫•n ƒë·ªÅ "Overlapping Subproblems").

```cpp
/**
 * @brief Ham de quy tinh gia tri C(n, k)
 * (Gia tri tai hang n, cot k)
 * @param n Hang (row)
 * @param k Cot (column)
 * @return Gia tri tai (n, k)
 */
long long TinhGiaTri_DeQuy(int n, int k) {
    // --- Phan co so (Base Case) ---
    // Neu la canh trai (k=0) hoac canh phai (k=n), gia tri la 1
    if (k == 0 || k == n) {
        return 1;
    }
    
    // --- Phan de quy (Recursive Step) ---
    // Gia tri = (tren-trai) + (tren-phai)
    return TinhGiaTri_DeQuy(n - 1, k - 1) + TinhGiaTri_DeQuy(n - 1, k);
}

/**
 * @brief In tam giac Pascal (De quy - Recursive)
 * @param n So dong can in
 */
void InPascal_Recursive(int n) {
    // Duyet qua tung hang (i tu 0 den n-1)
    for (int i = 0; i < n; i++) {
        
        // In ra cac khoang trang de can le dau hang
        for (int k = 0; k < (n - 1 - i); k++) {
            cout << "   ";
        }
        
        // Duyet qua tung phan tu trong hang i (j tu 0 den i)
        for (int j = 0; j <= i; j++) {
            // Goi ham de quy de lay gia tri
            cout << setw(6) << TinhGiaTri_DeQuy(i, j);
        }
        cout << endl; // Xuong dong khi in xong 1 hang
    }
}
```
