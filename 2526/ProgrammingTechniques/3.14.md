### 1\. ‚è© Ph∆∞∆°ng ph√°p Kh√¥ng ƒê·ªá quy (Iterative / V√≤ng l·∫∑p)

Ch√∫ng ta s·∫Ω d√πng m·ªôt v√≤ng l·∫∑p `while` ƒë·ªÉ li√™n t·ª•c thay th·∫ø `a` b·∫±ng `b` v√† `b` b·∫±ng s·ªë d∆∞ (`a % b`), cho ƒë·∫øn khi `b` b·∫±ng 0.

```cpp

 * @brief Tim USCLN (Khong de quy - Iterative)
 * Su dung Thu·∫≠t to√°n Euclid.
 * @param a So nguyen duong thu nhat
 * @param b So nguyen duong thu hai
 * @return USCLN cua a va b

int GCD_Iterative(int a, int b) {
    // Lap lai trong khi b chua bang 0
    while (b != 0) {
        // Tinh so du r
        int r = a % b;
        
        // Cap nhat gia tri cho vong lap tiep theo
        a = b;
        b = r;
    }
    
    // Khi b = 0, a chinh la USCLN
    return a;
}
```

### 2\. üîÑ Ph∆∞∆°ng ph√°p ƒê·ªá quy (Recursive)

 * @brief Tim USCLN (De quy - Recursive)
 * Su dung Thu·∫≠t to√°n Euclid.
 * @param a So nguyen duong thu nhat
 * @param b So nguyen duong thu hai
 * @return USCLN cua a va b

int GCD_Recursive(int a, int b) {
    
    // --- Phan co so (Base Case) ---
    // Neu b = 0, chung ta da tim thay ket qua, do la a
    if (b == 0) {
        return a;
    }

    // --- Phan de quy (Recursive Step) ---
    // Goi lai ham voi USCLN(a, b) = USCLN(b, a % b)
    return GCD_Recursive(b, a % b);
}
